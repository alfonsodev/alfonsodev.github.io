# Notes
This are the simplified instructions to install and run the website,
for a more detail 

## Requeriments
ruby 2.5.1 and bundler

## Install dependencies
clone this repo
```
cd notes 
bundler install
```

## Build the website
```
jekyll build
```
Alternatively you can use --watch to automatically rebuild the website.
